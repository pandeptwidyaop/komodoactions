# Usage Examples for Komodo Actions

After publishing the action to `github.com/pandeptwidyaop/komodoactions`, here's how to use it in other repositories.

## Quick Reference

**Minimal Setup (3 steps):**

1. Add secrets: `KOMODO_URL`, `KOMODO_API_KEY`, `KOMODO_API_SECRET`
2. Create `.github/workflows/deploy.yml`
3. Push!

See examples below for various use cases.
